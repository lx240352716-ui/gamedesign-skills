# 冰火双世界 PVE — 战斗策划详细设计

> 战斗策划 match 步骤输出  
> 参考配表: BeastPiratesBoss / BeastPiratesBossSkill / FightBuff

---

## 一、核心机制：属性克制

### 1.1 元素护盾（新增 FightBuff 2条）

| buff名 | 效果 | 挂载对象 | timing | 持续 |
|---------|------|---------|--------|------|
| 冰元素护盾 | 非火属性攻击伤害 -90% | 冰世界全体怪物 | 1(FightStart) | 永久 |
| 火元素护盾 | 非冰属性攻击伤害 -90% | 火世界全体怪物 | 1(FightStart) | 永久 |

实现参考：案例6（阵法 FightBuff），target=0(双方全体)，buff 挂在怪物方。

### 1.2 设计意图

- 玩家**必须**用火属性角色（艾斯）打冰世界，冰属性角色（大和）打火世界
- 切换世界 = 切换上阵阵容，增加策略深度
- 90% 减伤而非 100%，留 10% 容错给高战力玩家

## 二、冰世界（鬼岛·冰封区）

> 需要火属性角色（艾斯类）通关  
> 背景：百兽海贼团据守鬼岛冰封区，阻挡入侵者

### 第1关：霍金斯

| 项目 | 值 |
|------|------|
| 复用 bossId | 6 |
| Boss类型 | type=2（普通Boss） |
| 等级 | 120 |
| 怪组 | 复用 enemyGroupId |
| Boss技能 | **205(精准弱点)** + **934(元素武装)** |

**战斗设计**：

- 精准弱点(205)：精准攻击造成额外伤害，入门级弱点机制
- 元素武装(934)：自身附加元素伤害加成，增加输出压力
- 难度定位：**入门关**，让玩家熟悉属性克制机制
- 小怪：百兽海贼团 GIFTERS ×3（复用现有 enemyGroup 杂兵配置）

### 第2关：润媞

| 项目 | 值 |
|------|------|
| 复用 bossId | 5 |
| Boss类型 | type=2 |
| 等级 | 120 |
| Boss技能 | 203(元素弱点) + **924(虚掩大物)** |

**战斗设计**：

- 元素弱点(203)：对非克制属性角色额外减伤
- 虚掩大物(924)：Boss 自身获得护盾/减伤效果，需要集火击破
- 难度定位：**中等**，考验玩家集火策略
- 小怪：百兽海贼团精英 GIFTERS ×2 + 普通 GIFTERS ×2

### 第3关（终极Boss）：奎因

| 项目 | 值 |
|------|------|
| 复用 bossId | 2 |
| Boss类型 | type=1（精英敌人） |
| 等级 | 120 |
| Boss技能 | 204(屏障弱点) + **944(火焰之王)** + **953(属性免疫)** |

**战斗设计**：

- 屏障弱点(204)：Boss 自带屏障，需要特定方式击破
- 火焰之王(944)：Boss 周期性释放全体火焰伤害
- 属性免疫(953)：Boss 进入阶段2后短暂免疫属性伤害（需等免疫结束再输出）
- **阶段机制**：
  - 阶段1（HP 100%-50%）：屏障弱点 + 火焰之王，正常战斗
  - 阶段2（HP < 50%）：触发属性免疫 3回合 → 免疫结束后进入虚弱期 2回合（受伤 +50%）
- 难度定位：**困难**，需要把控输出节奏
- 小怪：百兽海贼团精英 ×4

## 三、火世界（鬼岛·烈焰区）

> 需要冰属性角色（大和类）通关  
> 背景：百兽海贼团据守鬼岛烈焰区，大和的反抗之路

### 第4关：佩吉万

| 项目 | 值 |
|------|------|
| 复用 bossId | 3 |
| Boss类型 | type=2 |
| 等级 | 120 |
| Boss技能 | **202(元素弱点)** + **913(十重真火)** |

**战斗设计**：

- 元素弱点(202)：对非克制属性角色额外减伤
- 十重真火(913)：对单体造成多段火属性伤害
- 难度定位：**入门关**（火世界侧）
- 小怪：百兽海贼团 GIFTERS ×3

### 第5关：德雷克

| 项目 | 值 |
|------|------|
| 复用 bossId | 4 |
| Boss类型 | type=2 |
| 等级 | 120 |
| Boss技能 | 202(元素弱点) + **923(真实破击)** |

**战斗设计**：

- 元素弱点(202)：克制机制
- 真实破击(923)：无视防御的穿透伤害
- 难度定位：**中等**，需要带奶抗住穿透伤害
- 小怪：百兽海贼团精英 GIFTERS ×2 + 普通 ×2

### 第6关（终极Boss）：烬

| 项目 | 值 |
|------|------|
| 复用 bossId | 1 |
| Boss类型 | type=1（精英敌人） |
| 等级 | 120 |
| Boss技能 | 206(扩散弱点) + **973(十重真火)** + **954(强韧之躯)** |

**战斗设计**：

- 扩散弱点(206)：伤害扩散到周围目标，需注意站位
- 十重真火(973)：多段火属性全体伤害（烬的火龙特性）
- 强韧之躯(954)：Boss 韧性极高，不易被控制
- **阶段机制**：
  - 阶段1（HP 100%-50%）：扩散弱点 + 十重真火，高频火焰输出
  - 阶段2（HP < 50%）：触发强韧之躯 → 攻击力 +30%、防御力 +20%、免控
  - 阶段2 特殊机制：每 3 回合释放一次「火龙焚天」全体大招（复用十重真火强化版）
- 难度定位：**最高难度**，DPS 赛跑 + 生存压力
- 小怪：百兽海贼团精英 ×4
